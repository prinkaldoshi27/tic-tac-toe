# Tic Tac Toe

A classic Tic Tac Toe game built with React, featuring an interactive game board, turn-based gameplay, winner detection, and game reset functionality.

The project demonstrates fundamental React concepts such as state management, component-based architecture, event handling, and conditional rendering through a simple and engaging game.

## 🎮 Features

- Two-player gameplay
- Interactive 3×3 game board
- Winner detection
- Draw detection
- Turn tracking (X and O)
- Game restart functionality
- Responsive user interface
- Real-time game updates

## ⚙️ Prerequisites

Before running the project, ensure you have:

- Node.js (v18+ recommended)
- npm

Verify installation:

```bash
node -v
npm -v
```

## 📥 Installation

Clone the repository:

```bash
git clone https://github.com/prinkaldoshi27/tic-tac-toe.git
```

Navigate to the project directory:

```bash
cd tic-tac-toe
```

Install dependencies:

```bash
npm install
```

## ▶️ Running the Application

Start the development server:

```bash
npm start
```

The application will run at:

```text
http://localhost:3000
```

## 🎯 Game Rules

1. The game is played on a 3×3 grid.
2. Two players take turns placing their symbol:
   - Player 1 → X
   - Player 2 → O
3. The first player to align three symbols horizontally, vertically, or diagonally wins.
4. If all squares are filled without a winner, the game ends in a draw.
